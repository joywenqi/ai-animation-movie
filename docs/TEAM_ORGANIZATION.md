## 职责矩阵 (RACI)

### 关键活动

| 活动/决策 | PM | 技术主管 | 创意主管 | QA | CEO | CFO |
|----------|----|---------|---------|----|-----|-----|
| 项目计划 | R | C | C | I | A | I |
| 技术方案 | C | R | - | C | A | I |
| 创意方案 | C | - | R | C | A | - |
| 进度控制 | R | C | C | C | A | I |
| 质量管理 | C | C | C | R | A | - |
| 成本控制 | R | C | C | C | A | R |
| 风险管理 | R | C | C | C | A | C |
| 人员调整 | C | A | A | C | A | - |
| 需求变更 | R | C | C | C | A | I |
| 重大决策 | I | I | I | I | R | C |

**图例**: R=负责 A=批准 C=咨询 I=知情

## 权力与权限

### 决策权

**PM** (项目经理)
- ✓ 日常工作安排
- ✓ 小范围资源调配
- ✓ <5万元支出审批
- ✓ 普通问题解决

**技术主管**
- ✓ 技术选型与方案
- ✓ 系统配置与部署
- ✓ 技术团队人事
- ✓ 技术工具采购

**创意主管**
- ✓ 创意方向决策
- ✓ 素材质量标准
- ✓ 创意团队调度
- ✓ 创意工具采购

**CEO**
- ✓ 所有重大决策
- ✓ >10万元支出
- ✓ 人员调整
- ✓ 外部沟通

## 工作时间与考核

### 工作制

- **标准工时**: 周一至周五，每天8小时
- **加班政策**: 需经PM同意，月加班<40小时
- **远程工作**: 核心工作需到办公室，其他可远程
- **休假政策**: 按国家规定执行

### 考核指标

#### PM 考核
- 进度完成度 (40%)
- 成本控制 (20%)
- 质量评分 (20%)
- 团队评价 (20%)

#### 技术主管 考核
- 技术指标完成 (40%)
- 团队产出 (30%)
- 创新贡献 (20%)
- 团队建设 (10%)

#### 创意主管 考核
- 创意质量 (40%)
- 进度完成度 (30%)
- 团队管理 (20%)
- 创新突破 (10%)

#### 工程师/设计师 考考核
- 工作产出 (50%)
- 代码/作品质量 (30%)
- 团队协作 (20%)
